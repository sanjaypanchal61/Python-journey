# Day 11 – f-string & Docstring

## 📖 f-string Definition

An f-string is used to format strings by embedding variables inside `{}`.

---

## 📖 Docstring Definition

A docstring is used to describe a function, class, or module.

---

## ⭐ Important Points

* f-strings improve readability
* f-strings use {}
* Docstrings use triple quotes
