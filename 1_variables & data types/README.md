## Day 1 - Variables & Data Types

### Variables

Variables are used to store data in a program.

Example:

```
name = "jack"
age = 20
```

### Data Types in Python

Python has different data types.

1. Integer (int)

```
age = 20
```

2. Float

```
price = 99.99
```

3. String

```
name = "jack"
```

4. Boolean

```
isStudent = True
```
