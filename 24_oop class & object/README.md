# Day 24 – OOP (Class, Object, Constructor)

---

## 📖 Definition

OOP (Object-Oriented Programming) is a programming paradigm based on **classes and objects**.

---

## 🏗️ Class

A class is a blueprint used to create objects.

---

## 🧍 Object

An object is an instance of a class.

---

## ⚙️ Constructor

Constructor is a special method that runs automatically when an object is created.

---

## 🧠 self Keyword

* Refers to the current object
* Used to access variables and methods

---

## ⭐ Important Points

* Class = blueprint
* Object = instance
* `__init__` = constructor
* `self` = current object

---

## 💡 Best Practices

* Use meaningful class names (PascalCase)
* Keep methods small and focused
* Use constructor for initialization

---
