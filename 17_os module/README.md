# Day 17 – OS Module

---

## 📖 Definition

The **OS module** is used to interact with the operating system for performing tasks like file handling, directory management, and system operations.

---

## ⭐ Important Points

* OS module is used for **system-level operations**
* Works on **Windows, Linux, Mac**
* `os.path` helps in safe file path handling

---

### Difference between os.mkdir() and os.makedirs()

* `mkdir()` → single folder
* `makedirs()` → multiple nested folders

---

## ⚠️ Common Mistakes

* ❌ Not checking file exists before deleting
* ❌ Using hardcoded paths (bad practice)
