## Day 6 - Match Case Statement

### Definition

The match-case statement is used to match a variable against multiple patterns and execute the corresponding block of code.

It works similar to the switch statement in other programming languages.

---

### Important Points

* Used for multiple conditions
* `_` works as default case

---

### When to Use Match Case

* One variable needs to be compared with many values.
* You want clean and readable conditional code.
