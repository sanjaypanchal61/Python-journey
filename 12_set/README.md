# Day 12 – Set

## 📖 Definition

A **set** is a collection of unique and unordered elements.

---

## Important Points

* Sets store unique values
* Unordered
* Cannot access elements using index
* Duplicate values are automatically removed

---

## ⚙️ Important Methods

* numbers.add()        # add element
* numbers.remove()     # remove element (error if not found)
* numbers.discard()   # remove without error
* numbers.clear()       # remove all elements
