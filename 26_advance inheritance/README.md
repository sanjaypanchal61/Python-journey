# Day 26 - Inheritance (Advanced Concepts)

## ✅ Topics Covered

* Multilevel Inheritance
* Hierarchical Inheritance
* Hybrid Inheritance

---

## 🔥 Multilevel Inheritance

### 🧠 Definition:

Multilevel inheritance is when a class is derived from another derived class.

### Explanation:

Child class can access properties of Parent and Grandparent.

---

## 🌳 Hierarchical Inheritance

### 🧠 Definition:

When multiple child classes inherit from a single parent class.

### Explanation:

Both Child1 and Child2 share the same parent class.

---

## 🔀 Hybrid Inheritance

### 🧠 Definition:

Hybrid inheritance is a combination of multiple and multilevel inheritance.

### Explanation:

Class D inherits from both B and C, forming a hybrid structure.
