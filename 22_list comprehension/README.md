# Day 22 – List Comprehension

---

## 📖 Definition

List comprehension is a concise way to create lists using a single line of code.

---

## ⚠️ Common Mistakes

* ❌ Writing very complex expressions
* ❌ Poor readability
* ❌ Using where simple loop is better

---

## 💡 Best Practices

* Keep it simple and readable
* Use for small transformations
* Avoid deeply nested comprehensions
