# Day 31 - Walrus Operator

---

## What is Walrus Operator?

The Walrus Operator (`:=`) allows assignment and expression in a single line.

---

## Syntax

```python
variable := expression
```

---

## Rules

* Use parentheses for clarity
* Avoid overuse
* Keep code readable

---

## Advantages

* Reduces code length
* Improves efficiency
* Useful in loops and conditions
