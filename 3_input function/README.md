## Day 3 - Input Function

### Definition

The input() function is used to take input from the user during program execution.

### Important Points

* The `input()` function waits for the user to enter a value.
* The value entered by the user is stored in a variable.
* By default, `input()` always returns data as a **string**.
* If we want numeric input, we must use **type casting** like `int()` or `float()`.
