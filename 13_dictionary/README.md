# Day 13 – Dictionary

---

## 📖 Definition

A **dictionary** is a collection of data stored in key-value pairs.

---

## 🧩 Important Methods

* student.keys()      # all keys
* student.values()    # all values
* student.items()     # key-value pairs
* student.clear()     # remove all

---

### Difference between get() and []

* `dict[key]` → return error if key not found 
* `dict.get(key)` → none return

---

## ⭐ Important Points

* Dictionary stores data in **key-value pairs**
* Keys must be **unique**
* Dictionary is **mutable**
* Access using **key, not index**
