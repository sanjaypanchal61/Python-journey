# Day 28 - Dunder Methods

## What are Dunder Methods?

Dunder methods are **special methods in Python** that start and end with double underscores.

---

👉 Example:

* `__init__`
* `__str__`
* `__len__`
* `__getitem__`

---

## Key Points

* Automatically called by Python
* Important for writing clean and powerful code

---

## Common Mistakes

❌ Forgetting to return string in `__str__`
❌ Using dunder methods unnecessarily

---

## Conclusion

👉 They improve flexibility and readability
👉 Python automatically triggers them
👉 Integrate with built-in functions
