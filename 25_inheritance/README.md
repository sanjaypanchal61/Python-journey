# Day 25 – Inheritance

---

## 📖 Definition

Inheritance allows a class (child) to acquire properties and methods of another class (parent).

---

## 🧠 Why Use Inheritance?

* Code reuse
* Cleaner structure
* Avoid repetition

---

## 🔥 Types of Inheritance

* Single Inheritance
* Multiple Inheritance

---

## ⭐ Important Point

* Parent → base class
* Child → derived class

---

## 💡 Best Practices

* Avoid deep inheritance chains
* Use meaningful class names
* Keep hierarchy simple
