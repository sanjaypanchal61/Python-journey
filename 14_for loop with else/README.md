# Day 14 – for loop with else

---

## 📖 Definition

The `else` block in a for loop executes only when the loop completes without a break.

---

## 🧠 How It Works

* Python first runs the `for` loop
* If the loop completes all iterations → `else` block runs
* If the loop is stopped using `break` → `else` block is skipped

---

## ⭐ Important Points

* `else` runs only if loop completes
* `break` stops the else block
* `continue` does NOT affect `else`
