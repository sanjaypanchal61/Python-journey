# Day 27 - super()

## 📌 What is super()?

The `super()` method is used to call methods or constructors from the parent class.

---

## 🔥 Why use super()?

* To reuse parent class code
* To call parent constructor

---

## ⚠️ Important Notes

* `super()` follows MRO (Method Resolution Order)
* Always use super() in multiple inheritance to avoid conflicts

---

## 🚀 Conclusion

The `super()` method is essential for:

* Clean code
* Reusability
* Proper inheritance handling
